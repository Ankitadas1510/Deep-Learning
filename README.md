# Deep Learning Image Classification using TensorFlow

## Description
This project implements a Convolutional Neural Network (CNN) using TensorFlow
to classify images from the CIFAR-10 dataset.

## Dataset
CIFAR-10 (built-in TensorFlow dataset)

## Technologies Used
- Python
- TensorFlow
- Keras
- Matplotlib

## Output
- Trained CNN model
- Accuracy visualization graph
